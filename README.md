# Ansible-Role-JDK
Super Simple Role to install Java Development 8 kit in linux.

Role Variables
--------------
```
Tools:
  - java-common
  - oracle-java8-installer
  - ca-certificates
  - oracle-java8-set-default
```

Example Playbook
----------------

    - hosts: YOUR-HOST-OR-GROUP
      roles:
    	- Ansible-Role-JDK
